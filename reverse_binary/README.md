# Reversed Binary

Problem ID: `reversebinary`

Difficulty: `Reggae`

## Usage

Programatically:

```ruby
require 'reverse_binary'

13.reverse # => 11
47.reverse # => 61
```

The `reversebinary` script can also used to supply a number on `stdin` and see the result on `stdout`.

    echo "13" | bin/reversebinary
    => 11
